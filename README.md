# Login and Register Animated form

This is an animated login and register form

## Tech and Librery

- React
- Vite
- TypeScript
- Tailwind css
- framer motion
- sonner
- zod

## Installation and Uses

```bash
git clone https://github.com/rahulroy0322/auth.git

cd auth

npm i
```

## Local Develpment / Run

run the project localy

```bash
npm run dev
```
